verilogify(func: function, namespace: dict[function, Context] = {}, module_output: PathLike | IO | None = None, testbench_output: PathLike | IO | None = None, write: bool = False, overwrite: bool = False)[source]¶
Parameters:

  • namespace – the namespace to put this function, for linking purposes

  • write – if True, files will be written to the specified paths

  • overwrite – If True, existing files will be overwritten

  • module_output – path to write verilog module to, defaults to function_name.sv

  • testbench_output – path to write verilog testbench to, defaults to function_name_tb.sv
